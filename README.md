Steps:

1. clone repo
1. `cd ./path/to/vscode-untitledactions`
1. `yarn`
1. `code .` or `code-insiders .`
1. `ctrl`/`cmd`+`shift`+`b` - wait for build to finish
1. `F5` (to launch the extension)
1. Open untitled empty file
